# Objetivo
Testes com módulo de áudio utilizando a biblioteca VLC.

## Packages
Instale os pacotes na raiz do seu projeto:
``` sh
    npm install
```

## Passos
1. Rode o projeto com:
``` sh
    npm run dev
```

2. No Postman, realize os testes de upload de um arquivo:

```
    localhost:5000/api/play_sound
```
